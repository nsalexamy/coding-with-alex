# Git 

- git clone The git clone command downloads an existing Git repository from a remote server onto your local machine
- git pull The git pull command downloads changes from a remote repository and immediately integrates them into your current local branch
- git push The git push command uploads your local repository commits to a remote repository, such as GitHub or GitLab
- git add The git add command moves file modifications from your working directory into the staging area (index)
- git commit The git commit command captures a permanent snapshot of your currently staged changes in your local Git repository
- git stash git stash temporarily shelves uncommitted changes (both staged and unstaged) so you can work on a clean directory, then restores them later
- git checkout git checkout is a Git command used to switch between branches, look at older commits, or discard local changes in your files
